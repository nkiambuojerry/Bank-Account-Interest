Bank Account Interest Calculation

This project calculates a bank account balance over a 12-month period, with monthly interest applied.

🔹 Project Features

  • ✅ Secure PIN authentication (3 attempts)
  • ✅ Interest calculation compounded monthly
  • ✅ Displays final balance after 12 months

📌 How to Run the Project

1. Clone the repository:
   git clone https://github.com/YOUR_GITHUB_USERNAME/Bank-Account-Interest.git
   cd Bank-Account-Interest

2. Run the Python script:
   python3 BankAccount.py
